DCC Beyond

Welcome to DCC Beyond, a character sheet manager for Dungeon Crawl Classics.
This current version is set up to import a json list of level 0 characters from the Purple Sorcerer level 0 character generator.
Choose JSON List as the Style/Format option. Then upload the file here.
Once you have a file uploaded you can edit any of the fields present. If you would like to save your changes click the Save Sheets button. You can specify a filename for the file.
For future sessions you can upload the modified file here to pick up where you left off.
